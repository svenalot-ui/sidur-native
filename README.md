# Сидур

Премиальное нативное iOS-приложение — спутник ежедневной еврейской молитвы.
SwiftUI, iOS 17+, дизайн «E-ink / editorial»: бумага, чернила, одна золотая линия.

## Что умеет

- **Зманим** — собственный солнечный движок (NOAA, сходится с KosherJava до минуты)
  + онлайн-уточнение через myzmanim. Все варианты расчётов (Гра, МА, Рабейну Там…),
  напоминания о временах дня.
- **Молитвы** — полные тексты Шахарит / Минха / Маарив по выбранному нусаху
  (Ашкеназ, Сфард, Хабад, Эдот а-Мизрах) из Сефарии, с офлайн-кэшем,
  навигацией по разделам и режимом погружения.
- **Брахот и личные молитвы** — иврит / транслит / перевод, из бандла
  (`Сидур/Content/liturgy.json` — см. [КАК_ПРИСЛАТЬ_ТЕКСТЫ.md](КАК_ПРИСЛАТЬ_ТЕКСТЫ.md)).
- **Теилим** — вся книга, по дням месяца, сгулот; закладки и «продолжить чтение».
- **Шаббат и Йом-Тов** — приложение **не работает** от зажигания свечей до исхода:
  экран покоя со свечами и временем исхода. Даты праздников — Hebcal
  (Израиль/диаспора по таймзоне), офлайн-кэш. Уведомления в дни покоя не приходят.
- **Напоминание о свечах** — за 30 минут до зажигания (Шаббат и Йом-Тов).
- **Мизрах** — компас на Иерусалим; **календарь** — еврейский месяц с событиями;
  **цдака** — реквизиты с QR.
- Два языка интерфейса (русский / иврит, RTL), авто-тема день/ночь по зманим,
  Dynamic Type.

## Сборка

Открыть `Сидур.xcodeproj` в Xcode 26+, собрать таргет «Сидур».
Файлы подхватываются автоматически (folder-synchronized group) — новые .swift
достаточно положить в папку `Сидур/`.

**Секреты:** ключ myzmanim лежит в `Сидур/Secrets.swift` (в .gitignore).
Без него приложение работает на собственном движке зманим.

## Структура

| Файл | Назначение |
|---|---|
| `AppState.swift` | язык, тема, нусах, локация, зманим, режим покоя (Шаббат/Йом-Тов) |
| `SolarTime.swift` / `Zmanim.swift` | солнечный движок и расчёт зманим |
| `MyZmanim.swift` | онлайн-уточнение зманим |
| `HolidayService.swift` | даты Йом-Това (Hebcal, офлайн-кэш) |
| `PrayerService.swift` | тексты служб из Сефарии по нусаху |
| `Liturgy.swift` + `Content/liturgy.json` | брахот и личные молитвы (бандл) |
| `Tehillim.swift` | Теилим: текст, циклы, сгулот |
| `ShabbatView.swift` | экран покоя (Шаббат/Хаг) |
| `Theme.swift` / `Components.swift` | палитра, типографика, общие контролы |
| `Notifications.swift` | напоминания о зманим и свечах |

## Дизайн

Архетип — «E-ink + editorial»: тёплая бумага `#FCFAF5`, чернила, золото `#986413`
только для действия и одной фирменной линии. Шрифты: Playfair Display (кириллица,
заголовки), Frank Ruhl Libre (иврит), Bodoni Moda (цифры), SF Mono (микро-ярлыки).
